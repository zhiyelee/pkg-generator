# pkg-generator
Generate  NPM Package

## Installation

> npm install pkg-generator -g

## Usage

```bash
# create a package directory named `mds` in current directory
pkg-generator mds
```
which produces directory as follows,

``` 
✍ tree -a mds
mds
├── .npmignore
├── .travis.yml
├── LICENSE
├── README.md
└── package.json

0 directories, 5 files
```
